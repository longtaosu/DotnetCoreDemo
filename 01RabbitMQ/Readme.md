# 说明

.Net下RabbitMQ常用的SDK有RabbitMQ.Client、EasyNetQ、RawRabbit

按照下载量：RabbitMQ.Client > EasyNetQ > RawRabbit



**说明：**

- RabbitMQ.Client是官方Demo给出的SDK，star及下载量是最多的，也是CAP、Surging等框架采用的SDK
- EasyNetQ对RabbitMQ.Client进行二次封装，操作更简单
